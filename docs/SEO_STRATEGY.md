## 3. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### 3.1 Улучшения robots.txt
```
User-agent: *
Allow: /
Disallow: /admin/
Disallow: /backups-server/
Disallow: /login/
Disallow: /register/

Sitemap: https://evrocontayner.kz/sitemap.xml
Sitemap: https://evrocontayner.kz/sitemap-cities.xml
```

### 3.2 Расширенный sitemap.xml
- Добавить `<lastmod>` для каждой страницы
- Добавить `<priority>` (1.0 для главной, 0.8 для основных, 0.6 для городов)
- Добавить `<changefreq>` (weekly для основных, monthly для городов)
- Создать отдельный sitemap для городов (sitemap-cities.xml)
- Создать sitemap для изображений (sitemap-images.xml)

### 3.3 Schema.org структурированные данные
Добавить на все страницы:
- **Organization** (на главной):
  - Название: Evrocontayner
  - Лого, контакты, соцсети, адрес
  
- **LocalBusiness** (для каждого города):
  - Город, услуги, контакты
  
- **Product** (для каждого товара):
  - Название, описание, цена, изображения, отзывы
  
- **FAQPage** (для услуг и продуктов)
- **BreadcrumbList** (для навигации)

### 3.4 Производительность и Core Web Vitals
- [ ] LCP (Largest Contentful Paint) < 2.5s
- [ ] FID (First Input Delay) < 100ms
- [ ] CLS (Cumulative Layout Shift) < 0.1
- Использовать: image optimization, code splitting, lazy loading

### 3.5 Мобильная оптимизация
- ✅ Viewport meta tag добавлен
- [ ] Mobile-friendly дизайн (уже есть)
- [ ] Оптимизация размеров изображений
- [ ] Быстрая загрузка на медленных сетях (3G)

**Шаги:**
